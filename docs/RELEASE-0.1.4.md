# PMDO Android Player 0.1.4

Initial Android preview for ARM64 devices running Android 13 or newer, with a
focus on the AYN Thor's upper display.

## Highlights

- PMDO 0.8.12 runtime import through Android's system folder picker.
- Unmodified Lua/data/asset Quest mod folder and ZIP import.
- Physical controller input and a multi-touch gamepad overlay.
- Maximum 4:3 as the default, with full-screen and fixed 4:3 display modes.
- Android keyboard, clipboard, and save import/export support.
- Improved music recovery and overlapping sound-effect source management.
- **PMDO Android Player** app name and an original neutral launcher icon.

## Installation

- [General Android guide](https://github.com/solbounder/pmdo-android-player/blob/main/docs/INSTALL-ANDROID.md)
- [AYN Thor guide](https://github.com/solbounder/pmdo-android-player/blob/main/docs/INSTALL-AYN-THOR.md)

Install future versions directly over the existing app. Do not uninstall first,
because Android removes the private runtime, mods, and saves during uninstall.

The APK contains no PMDO runtime, Pokémon assets, Quest data, or saves.
