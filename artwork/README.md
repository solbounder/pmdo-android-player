# Artwork provenance

`app-icon-source.png` was generated specifically for this project with OpenAI
image generation on 2026-07-21 and then resized into the Android density
variants under `src/PMDO.Android/Resources`.

The icon is an original abstract portal design. It was generated without a
reference image and intentionally contains no Pokemon character, franchise
symbol, text, or third party logo.

Generation prompt:

> Create a clean original square app icon showing an abstract glowing portal
> with three subtle concentric echo rings above a small dark path leading
> inward. Use a polished minimalist digital icon style with a crisp readable
> silhouette at very small sizes, a deep navy background, teal and violet
> portal glow, and a small warm gold accent. Do not include characters,
> creatures, monster balls, letters, words, logos, trademarked symbols, or
> watermarks.
