# Echoes of the Abyss Android 0.1.5

Android preview update with a persistent, freely configurable touch overlay.
Open the gear while the overlay is visible, drag a control, long-press it for
binding/visibility/scale, then save, cancel, or reset. The default layout now
includes ZL and ZR. Physical controller behavior is unchanged.

L, R, ZL and ZR can each mark items. Any left/right shoulder combination
selects all sellable entries across inventory pages. Audio volumes and fanfare
fades are bounded to prevent the reported Android error/black-screen path.

The refreshed Thor update ZIP includes the reviewed Echoes of the Abyss 0.8.2
Quest tree, including accessible secret-room exits and the accepted Abyss
artwork converted to engine-native tiles for the dungeon-selection map.

This is an in-place update (`versionCode` 6); do not uninstall the app.
