PMDO ANDROID - UPDATE FOR AN EXISTING THOR INSTALLATION
======================================================

This update ZIP contains:

  PMDO-Android-Player-v0.1.9.apk
  THOR-UPDATE.txt

Do NOT uninstall the app. Updating it in place preserves the imported PMDO
runtime, optional mods, settings, and save data.

1. Copy the ZIP to "Internal storage\Download" on the Thor and extract it.
2. Tap the APK and confirm "Update" or "Install".
3. Open PMDO Android Player and tap "Start game". The runtime, optional mods,
   and save data do not need to be imported again.

New in this update:

  - Browsing the PMDO music menu no longer reserves an Android audio source for
    every listed track. This removes the source-exhaustion cause identified in
    the reported exception and subsequent audio lockup.
  - All launcher, status, dialog, error-report, keyboard, and touch-editor text
    owned by the Android app is now English.
  - Save permission errors now explain how to recover from files copied
    manually with incompatible Android ownership.

Quest mods are not included in this player package. They can be added
separately through "Import mod folder" or "Import mod ZIP".

If Android reports a signature conflict instead of offering an update, do NOT
uninstall the existing app. Capture the dialog so the installed private data
remains untouched.
